# 📊 电商用户行为数据分析项目

## 📌 项目简介

本项目基于电商平台用户行为数据（浏览、点击、加购、购买），通过数据清洗、转化漏斗分析与可视化建模，对用户行为路径进行分析，识别关键流失环节，并提出优化建议。

该项目完整覆盖数据分析流程，适用于数据分析岗位求职展示。

---

## 🎯 项目目标

* 分析用户行为路径（view → click → cart → buy）
* 构建转化漏斗模型，计算各阶段转化率
* 分析用户活跃情况（DAU）
* 统计用户行为分布特征
* 提供数据驱动的业务优化建议

---

## 🛠 技术栈

* Python（Pandas / NumPy）
* 数据可视化（Matplotlib）
* Jupyter Notebook

---

## 📂 数据说明

数据为模拟生成的电商用户行为数据，字段如下：

| 字段名      | 含义                              |
| -------- | ------------------------------- |
| user_id  | 用户ID                            |
| behavior | 用户行为（view / click / cart / buy） |
| time     | 行为发生时间                          |

---

## 🔍 分析内容

### 1️⃣ 转化漏斗分析（核心）

构建用户行为转化路径：

view → click → cart → buy

计算各阶段转化率：

* 点击率 = click / view
* 加购率 = cart / click
* 购买率 = buy / cart

👉 用于识别用户流失最严重的环节

---

### 2️⃣ 用户活跃分析

* 统计每日活跃用户（DAU）
* 分析用户活跃趋势变化

---

### 3️⃣ 用户行为分布分析

* 统计各类行为占比
* 分析用户主要行为阶段

---

## 📈 可视化展示

---

## 📊 分析结论

* 用户从浏览到购买存在明显流失
* “加购 → 购买”阶段流失最严重，是优化重点
* 用户行为主要集中在浏览阶段
* 用户活跃度存在周期性波动

---

## 💡 优化建议

* 简化支付流程，减少购买阻力
* 提供优惠活动，提高转化率
* 优化商品详情页，提高点击率
* 针对高活跃用户进行精准营销
